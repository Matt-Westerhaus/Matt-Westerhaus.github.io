Optum Internship:

This past summer I worked with a team of interns to create an open source Python library designed to help people with data preprocessing. It relies on functionality from the already defined libraries Pandas, NumPy, Sci-kit Learn and I also was able to familarize myself with Keras and Tensorflow. The library that we made, named Soapi to reflect that it "cleans" data, holds functions to drop insignificant columns and rows, fill in missing values if desired, transforming categorical values to numerical values to speed up machine learning model training time along with a function to reverse this once the model is complete and many functions to gain information from your data.
